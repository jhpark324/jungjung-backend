# Backend Scripts

이 디렉토리에는 유틸리티 스크립트가 포함되어 있습니다.

## visualize_graph.py

LangGraph 그래프를 시각화하는 스크립트입니다.

### 실행 방법

```bash
cd /Users/parkjehyeong/jungjung
uv run python -m backend.scripts.visualize_graph
```

### 출력

- `backend/docs/rag_graph.png` - RAG 그래프 이미지
- `backend/docs/agent_graph.png` - Agent 그래프 이미지  
- `backend/docs/rag_graph.md` - RAG 그래프 Mermaid
- `backend/docs/agent_graph.md` - Agent 그래프 Mermaid
